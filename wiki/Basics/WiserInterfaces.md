# Wiser Interfaces

Wiser Interfaces are special built-in modules in Wise. They configure themselves automatically across all your characters, so you don't have to manually set up common actions.

## Available Wiser Interfaces

### 1. Professions
Auto-populates a ring (or whatever layout you choose) with your current character's professions and secondary skills (Fishing, Cooking, Archaeology). It generates a toggle macro for each profession, allowing you to open and close the profession window with the same button.

### 2. Menu Bar
A replacement for the tiny default WoW micro-menu. It creates a customizable interface containing links to the Menu, Shop, Adventure Guide, Collections, Group Finder, Guild, Housing, Quest Log, Achievements, Spellbook, Talents, Professions, and Character Info. You can display it as a big clickable ring, list, or box.

### 3. Forms (Shapeshift / Stances)
Automatically detects your class's shapeshift forms or stances (e.g., Druid forms, Warrior stances, Rogue stealth) and creates buttons for them. It uses external skinning (like Masque) if available.

### 4. Specs
Provides a quick way to switch between your character's specializations without opening the talent pane. It automatically updates when you change specs or log into a different character.

## How to use Wiser Interfaces

1. Open the Wise Options Panel (`/wise`).
2. Wiser Interfaces are listed in the interface list on the left side, indicated by a special icon or color.
3. Click on a Wiser Interface to configure it.
4. While the **Actions** in a Wiser Interface are generated automatically, you can fully customize the **Settings** (Layout type, Icon Size, Visibility, Position, etc.) just like any other Wise interface.
5. If a Wiser Interface is disabled by default, check its Visibility settings to enable it. (See [Visibility Settings](Visibility.md)).

Wiser Interfaces dynamically update whenever your character's info changes (e.g., learning a new profession, changing specs).
